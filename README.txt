=========================================
SISTEMA DE SCRAPING Y CATÁLOGO AUTOMÁTICO
=========================================

DESCRIPCIÓN:
Este sistema automatizado extrae información en tiempo real de 
MercadoLibre (Zapatillas), descarga las imágenes y genera una 
plataforma web de visualización sin intervención humana.

TECNOLOGÍAS UTILIZADAS:
- Python 3.12 (Core)
- BeautifulSoup4 (Extracción de HTML)
- Requests (Gestión de red y descargas)
- CSV (Base de datos ligera)
- HTML5/CSS3 (Interfaz de usuario)

CÓMO USAR:
1. Ejecute 'PROYECTO_FINAL.py'.
2. El sistema creará la carpeta de fotos y el reporte Excel.
3. Al finalizar, abra 'tienda_automatica.html ubicada en el explorador de archivos' para ver el catálogo.
